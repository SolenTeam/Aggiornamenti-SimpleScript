# Aggiornamenti-SimpleScript

SimpleScript è un complesso di comandi rapidi per velocizzare parzialmente il tuo telefono o tablet Apple. 
Questa repository è stata creata sperimentalmente per aggiornamenti e nuove funzioni che si trovano sul [canale WhatsApp](https://whatsapp.com/channel/0029VbCXSQ03LdQNZgapAh3P)

All'avvio di SimpleScript verrà verificata la disponibilità degli aggiornamenti ed è questa la vera funzione della repository.
